# Security Policy

## Supported Versions

Only the latest version of Peyar Amudham is currently supported.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

We take the security of Peyar Amudham seriously. If you believe you have found a security vulnerability, please report it to us by practicing responsible disclosure.

Please do **not** report security vulnerabilities via public GitHub issues.

Instead, please report them to **hello@sivakaminathan.in**.

We will acknowledge receipt of your vulnerability report within 48 hours and will send you regular updates about our progress.
